# 🛍️ Anderson Market

**Marketplace d'accessoires premium au Congo Brazzaville avec intégration IA**

Une plateforme e-commerce moderne, professionnelle et intuitive pour acheter et vendre des accessoires de qualité.

## 🌟 Caractéristiques principales

- 🔍 **Recherche avancée** avec filtrage par catégorie, prix et vendeur
- ⭐ **Système de notation** pour les produits et vendeurs
- 🤖 **Recommandations IA** personnalisées
- 🖼️ **Galerie d'images** professionnelle avec zoom
- 📱 **Design responsive** pour tous les appareils
- ✅ **Profils vendeurs vérifiés**
- 📞 **Intégration WhatsApp** directe

## 🛠️ Stack technologique

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- React Router v7
- Axios
- Lucide React

## 📋 Installation

```bash
npm install
```

### Frontend
```bash
npm run dev
```

### Backend
```bash
cd server
npm install
npm run dev
```

## 🔧 Configuration

Copier le fichier `.env` si besoin et configurer l'API :

```bash
cp .env.example .env
```

`VITE_API_URL=http://localhost:3001/api`

## 🚀 Build

```bash
npm run build
npm run preview
```
